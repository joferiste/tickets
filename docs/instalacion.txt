========================================
SISTEMA DE BOLETAS BANCARIAS - v1.0
INSTRUCCIONES DE INSTALACIÓN
========================================

REQUISITOS:
-----------
1. Windows 10/11 (64-bit)
2. Mínimo 8 GB RAM
3. 10 GB espacio en disco
4. Conexión a Internet (primera instalación)

INSTALACIÓN:
------------

PASO 1: INSTALAR DOCKER DESKTOP
================================
1. Descargar Docker Desktop desde:
   https://www.docker.com/products/docker-desktop

2. Ejecutar el instalador
   - Aceptar instalar WSL 2 si lo pide
   - Reiniciar la computadora si es necesario

3. Abrir Docker Desktop
   - La primera vez puede tardar 2-3 minutos en iniciar
   - Verás un ícono de ballena en la bandeja del sistema
   - Esperar a que diga "Docker Desktop is running"

PASO 2: CONFIGURAR EL SISTEMA
==============================
1. Copiar la carpeta completa "SistemaBoletasBancarias" a:
   C:\Aplicaciones\SistemaBoletasBancarias\

2. Ir a la carpeta "tickets"

3. Copiar el archivo ".env.example" y renombrarlo a ".env"

4. Abrir ".env" con Notepad y configurar:
   
   POSTGRES_PASSWORD=PonerPasswordSeguroAqui
   
   SECRET_KEY=GenerarNuevaSecretKey
   
   EMAIL_USERNAME=tu_correo@gmail.com
   EMAIL_PASSWORD=tu_app_password
   
   (Guardar y cerrar)

PASO 3: INICIAR EL SISTEMA
===========================
1. Hacer doble clic en:
   "Sistema de control administrativo.exe"

2. Primera vez: Seleccionar la carpeta "tickets"

3. Presionar "Iniciar Sistema"

4. Esperar 1-2 minutos (primera vez descarga imágenes)

5. El navegador se abrirá automáticamente

USUARIOS:
---------
La primera vez, crear un superusuario:

1. Abrir PowerShell en la carpeta del sistema
2. Ejecutar:
   cd tickets
   docker compose exec django python src/manage.py createsuperuser

3. Seguir las instrucciones en pantalla

SOPORTE:
--------
En caso de problemas:
1. Ver el archivo TROUBLESHOOTING.txt
2. Revisar logs en la carpeta "logs/"
3. Contactar a soporte técnico
